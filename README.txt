Тестовое задание для НТЦ ПРОТЕЙ.

Задание:
Написать DDT (data driven tests) тесты публичного API геокодинга openstreetmaps.org (проект nominatim).
Протестировать необходимо прямое (адрес -> координаты) и обратное (координаты -> адрес) геокодирование.

Использован ЯП Python 3.6.1
Сторонние библиотеки:
requests - для осуществления HTTP-запросов и получения ответов
xlrd - для чтения .xls файлов
urllib.parse - для URL-encoding - преобразование обычного текста в URL-вид
